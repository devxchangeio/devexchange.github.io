Java ORM Plugin for Eclipse

November, 2014
|
Eclipse Plugin

Java ORM Plugin for eclipse helps the user to create a java ORM framework in one click.

  • MyBatis ORM Framework
  • Hibernate ORM Framework
Requirements:
Compatible with Windows,Mac,Linux. Requires Eclipse 3.5 or later.
Downloadable:
ORM Plugin is available for Download in Eclipse Marketplace and Sourceforge.
